标题：

  • 标题除了可以通过#来设置,还可以通过 ctrl+数字键 1,2,3 来设置

列表:

无序列表

  • 语法很简单,只需要在你的文字前面加上一共-或*就行了.

有序列表

  • 在文字前加1. 2. 3. 的符号

注意：- * 等符号要和文字之间加一共空格

引用：

  • 只需要在文本在之前加入>这种尖括号

粗体与斜体

  • Markdown 的粗体和斜体也非常简单，用两个 * 包含一段文本就是粗体的语法，用一个 * 包含一段文本就是斜体的语法。

代码框

  • 连续三个反引号`，就是按键1前面的那个符号再加代码类型.
